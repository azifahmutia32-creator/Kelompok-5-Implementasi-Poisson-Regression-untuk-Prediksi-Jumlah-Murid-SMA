# Implementasi Poisson Regression untuk Prediksi Jumlah Murid SMA Berdasarkan Jumlah Sekolah dan Jumlah Guru di Indonesia Tahun 2025/2026

## Deskripsi Proyek

Proyek ini merupakan implementasi metode Poisson Regression untuk memprediksi jumlah murid SMA di Indonesia berdasarkan jumlah sekolah dan jumlah guru pada tahun ajaran 2025/2026. Analisis dilakukan menggunakan data pendidikan yang diperoleh dari Badan Pusat Statistik (BPS).

## Dataset

Dataset yang digunakan berjudul:

**Number of Schools, Headmasters and Teachers, and Pupils in Senior High Schools Under The Ministry of Primary and Secondary Education by Province, 2025/2026**

Variabel yang digunakan:

* Jumlah Sekolah SMA
* Jumlah Guru SMA
* Jumlah Murid SMA

## Metode

Metode yang digunakan dalam proyek ini adalah **Poisson Regression**, yang sesuai untuk memodelkan data cacahan (count data).

Tahapan analisis:

1. Data preprocessing
2. Pembersihan data (data cleaning)
3. Penanganan missing value
4. Feature scaling
5. Pembagian data training dan testing
6. Pelatihan model Poisson Regression
7. Evaluasi model menggunakan Mean Absolute Error (MAE)

## Hasil

Model berhasil digunakan untuk memprediksi jumlah murid SMA berdasarkan jumlah sekolah dan jumlah guru. Evaluasi model menghasilkan nilai MAE sebesar 84.671,56.
